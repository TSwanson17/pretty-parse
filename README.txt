This project requires the Stanford Core-NLP library.
The libriary can be downloaded at https://stanfordnlp.github.io/CoreNLP/
	Scroll down to Quickstart
	Download and unzip CoreNLP 4.4.0
	Download the English (KBP) language model (stanford-corenlp-4.4.0-models-english-kbp.jar)
		Place the jar file in the unziped CoreNLP folder (stanford-corenlp-4.4.0)
	Place the CoreNLP folder (stanford-corenlp-4.4.0) in the src folder
